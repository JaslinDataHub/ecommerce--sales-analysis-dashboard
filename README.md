#  E-Commerce Sales Analysis Dashboard | Power BI

##  Project Overview

This project presents an interactive E-Commerce Sales Analysis Dashboard developed using Microsoft Power BI.

The dashboard transforms raw sales data into meaningful insights related to sales performance, profit, customers, products, payment modes, and regional performance.

## Project Objective

The objective of this project is to analyze e-commerce sales data and create an interactive dashboard that helps in understanding business performance and supports data-driven decision-making.

##  Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Data Cleaning
- Data Analysis
- Data Visualization

##  Key Performance Indicators

- Total Sales: 438K
- Total Quantity: 6K
- Total Orders: 500
- Total Profit: 37K
- Profit Margin: 0.08

##  Dashboard Features

- Monthly sales trend analysis
- Sales analysis by product category
- Sales analysis by sub-category
- Top 10 customer analysis
- Payment mode analysis
- Profit analysis by category
- Profit analysis by city
- Top cities by sales
- Interactive quarter filter

## Key Insights

- The dashboard provides an overview of overall sales and profitability.
- Sales performance can be compared across different product categories.
- Customer and city-wise performance helps identify top-performing segments.
- Payment mode analysis provides insights into customer purchasing preferences.
- Monthly trends help understand changes in sales performance over time.

##  Project Outcome

This project demonstrates the use of Power BI for data cleaning, data analysis, KPI creation, and interactive data visualization.
